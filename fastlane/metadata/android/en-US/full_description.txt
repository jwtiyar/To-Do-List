To-Do: Task List & Reminder is a professional-grade task management application designed for simplicity and efficiency.

Features:
- Complete task management (CRUD operations)
- Task reminders with local notifications
- Encrypted Backup & Restore functionality
- Modern Material Design 3 interface
- Multiple themes (Light, Dark, and System Auto)
- Multi-language support
- Advanced search and filtering
- Fully offline and privacy-focused
- Open source (GPLv3)

Built with modern Android architecture (MVVM, Hilt, Room, Paging 3).
