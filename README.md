# To-Do: Task List & Reminder

A professional-grade, privacy-focused task management application for Android.

## Features

- **Task Management**: Simple and efficient CRUD operations.
- **Reminders**: Local notifications for your tasks.
- **Secure**: Fully offline with encrypted Backup & Restore.
- **Modern UI**: Material Design 3 with Light/Dark mode.
- **Privacy**: No tracking, no ads, no cloud required.

## Build

The app uses standard Gradle. You can build it using:

```bash
./gradlew assembleRelease
```
